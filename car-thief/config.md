# Konfiguracija

Sve opcije su u `config.lua`.

| Key          | Tip    | Default | Opis |
|--------------|--------|---------|------|
| `Locale`     | string | `"hr"`  | Jezik (`hr`/`en`). |
| `NotifyType` | string | `"esx"` | Naƒçin notifikacija (`esx`/`ox`/`chat`). |

### Primjer
```lua
Config.Locale = 'hr'
Config.NotifyType = 'ox'
```
